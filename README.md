Overview- About the MOOD TRACKER APP
The Mood Tracker Web App is a simple yet powerful platform designed to help users record, monitor, and reflect on their emotional well-being over time. It provides an intuitive interface for users to log daily moods, add notes, and view visual summaries through charts and graphs.
Users can identify emotional patterns, track triggers, and gain insights that promote mindfulness and personal growth. The application is built using Flask (Python) for the backend
